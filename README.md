# 🕸️ Webスクレイピング支援ツール

HTMLソースを解析・レンダリングし、要素情報やURL規則設定、スクレイピング結果を可視化するWebアプリケーションです。

## ✨ 機能

| 機能 | 説明 |
|------|------|
| **HTML入力・解析** | HTMLソースを貼り付け、危険なスクリプトを除去してプレビュー |
| **HTMLビューア** | iframe内でプレビュー、要素クリックで属性情報（class, id, data-*）を表示 |
| **URL生成** | テンプレートと数値範囲/リストからURL一括生成 |
| **robots.txt確認** | スクレイピング前にアクセス許可をチェック |
| **スクレイピング実行** | リアルタイムログ表示、結果を一覧表示 |

## 🛠️ 技術スタック

- **フロントエンド**: React + Vite + TailwindCSS
- **バックエンド**: Python FastAPI + BeautifulSoup4
- **開発環境**: Ubuntu 22.04 LTS

## 🚀 起動方法

### 1. バックエンド

```bash
cd backend

# 依存パッケージのインストール
pip install -r requirements.txt

# サーバー起動
uvicorn main:app --reload --port 8000
```

API ドキュメント: http://localhost:8000/docs

### 2. フロントエンド

```bash
cd frontend

# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev
```

アプリケーション: http://localhost:5173

## 📖 使い方

1. **HTML入力タブ**: スクレイピング対象のHTMLソースを貼り付け →「レンダリング」ボタン
2. **HTMLビューアタブ**: 要素をクリックして属性情報を確認
3. **URL生成タブ**: テンプレートとプレースホルダを設定 → URLリスト生成
4. **スクレイピングタブ**: robots.txt確認後、スクレイピング実行

## ⚠️ 注意事項

- スクレイピング前に必ず対象サイトの `robots.txt` を確認してください
- 過剰なリクエストを送らないよう、適切なリクエスト間隔を設定してください
- 本ツールは**教育・調査目的専用**です
